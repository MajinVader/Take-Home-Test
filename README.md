# Booking Lapangan API  
Backend Take Home Test – Sagara Asia Teknologi  

---

## Overview
API backend untuk sistem booking lapangan olahraga dengan fitur:

- User & Admin Authentication (JWT)
- CRUD Lapangan (Admin Only)
- Booking lapangan dengan pengecekan jadwal bentrok (overlap)
- Payment (mock) untuk mengubah status booking menjadi paid
- Role-based Authorization
- PostgreSQL + GORM ORM
- Fiber Framework (Golang)

---

## Tech Stack

### **Backend**
- Go 1.22+
- Fiber v2
- GORM ORM
- PostgreSQL (versi bebas: 14–18)
- JSON Web Token (JWT)
- Bcrypt password hashing

### **Tools**
- Postman 
- Docker 
- pgAdmin
